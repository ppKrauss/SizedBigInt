Class: SizedBigInt

SizedBigInt()

Sized BigInt's (SizedBigInt) are arbitrary-precision integers with defined number of bits. Each instance is a pair (size,number). Input and output accepts many optional representations. Original source at github.com/ppKrauss/SizedBigInt and foundations at osm.codes/_foundations/art1.pdf.
